Welcome to DB4Sci

Overview

DB4SCI allows faculty and staff to create a stand alone database container instance. It may house one database or many. The DB instances are created in docker containers on a shared server managed by Scicomp. You will not have direct access to the file system but you will have superuser permissions to the database instance through psql, mysql or other languages. Current support for these database servers is minimal. We will back up your databases, allowing for recovery with data loss of up to 24 hours. We can assist with tuning your database if performance is an issue.
