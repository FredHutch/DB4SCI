Demo Install

DB4SCI requires a lot of configuration. To get the application up and running quickly, a demo mode is available for Linux. Demo mode functions just like a production instance but is not secure for production use. Demo mode removes TLS encryption and authentication. Demo mode requires a Docker CE version 18 and docker-compose to be installed. Install can take 20 minutes or longer depending on your internet connection. The install script downloads all the Docker containers needed.
