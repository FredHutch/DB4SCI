Demo Install

DB4SCI requires a lot configuration. To get the application up and running quickly, a demo mode is available for Mac OSX. Demo mode removes Nginx, uWSGI, and https. Demo mode also removes authentication. Demo mode is intended for personal use on your Mac and should not be used in production.

Install Docker CE native on your Mac. https://store.docker.com/editions/community/docker-ce-desktop-mac Download and run the install script. Install can take 20 minutes or longer depending on your internet connection. The install script downloads all the Docker containers needed. When the install script is complete browse to this location: 172.16.123.1
