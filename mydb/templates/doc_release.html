Release Notes

Release 1.7.0 Target Feb 2018

  • Add support for Data Encrypt at Rest and TLS for MariaDB. Encrypt at Rest is an option on the MariaDB create form. TLS is enabled if encryption at rest is selected.
  • List S3 Backups by container name
  • CSS format changes

    Docker API Update

    DockerAPI has changed. Docker CE and EE are an evolution of the Docker Platform. CE is the Comunity Edition for docker and EE support Enterprise customers. Previously the Linux package docker.io was installed. docker.io and docker-py have been depreicated. Docker Python 2 API package is docker-3.0.x. The new API package is only available for download from Docker. 

     
      docker 1.3 -> 17.5 
      Upgrade docker.io to docker-ce
      upgrade docker-compose 1.18 -> 1.19.0
      Dockerfile
      requirement.txt

    The Docker client API has changed with docker Python version 2. The code has been refactored to move all Docker API calls to the container_util.py module. This is the new docker API client call:

    client = docker.APIClient(base_url='unix://var/run/docker.sock')
